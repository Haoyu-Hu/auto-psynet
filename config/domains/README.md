# Domain Design-Priors

The thinner, complementary layer of the Knowledge Pack (project-plan §4.0). Per behavioral domain:
typical constructs/measures, standard paradigms (via the affinity), common confounds/controls, ballpark
effect sizes, and key prior-work pointers. Consulted during FORMULATE. **Demand-driven** — seeded with
the user's subfields, expanded as needed (not an enumeration of all of behavioral science).

## Seeded domains (from kickoff D8)
- [perception](perception.md) — vision / audition / multisensory; thresholds, scaling, representation
  recovery. **(anchor)**
- `music_audio` — pitch, rhythm, consonance, timbre, prosody (PsyNet heritage). *stub*
- `language` — word learning, semantics, iterated language evolution, dialogue. *stub*
- `memory_learning_decision` — recall/recognition, category learning, judgment/choice, preferences. *stub*
