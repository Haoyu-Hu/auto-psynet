# Blind-Spot Library

Keyword-triggered, injectable checklists of "things behavioral experiments systematically get wrong."
Gates (especially G1) consult these so the methodologist/statistician personas don't miss known
pitfalls. Each file groups pitfalls by topic with a short trigger + the check to apply.

- [methodology.md](methodology.md) — design & statistical pitfalls (starter set).

Add files as patterns recur (e.g. `audio.md`, `cross-cultural.md`, `online-recruitment.md`).
