# Blind spots — methodology & statistics (starter)

Each item: **trigger** → **the check to apply**.

- **Confounds** — any manipulation → is there an uncontrolled variable that co-varies with the IV?
- **Demand characteristics** — within-subject / transparent manipulation → could participants guess the
  hypothesis and comply? Consider between-subjects or cover stories (with debrief).
- **Order / carryover effects** — repeated measures → is presentation order randomized / counterbalanced?
- **Ceiling / floor** — rating or accuracy tasks → will the measure saturate? pilot the range.
- **Underpowered design** — confirmatory claim → is N justified by a power analysis for the target effect?
- **Multiple comparisons** — many outcomes/tests → is multiplicity controlled? pre-specify primary outcome.
- **p-hacking / HARKing** — flexible analysis → is the analysis preregistered and held as a holdout (G1)?
  are exploratory analyses labeled as such?
- **Bot / bad-actor contamination** — online recruitment → attention checks, completion-time bounds,
  duplicate detection, and exclusion rules pre-specified.
- **Non-independence** — nested data (trials in participants in chains) or mixed human-AI samples → use
  the right hierarchical model; don't treat trials as independent.
- **Measurement invariance** — cross-cultural / multilingual comparison → test configural/metric/scalar
  invariance before comparing constructs across groups.
- **Synthetic ≠ human** — LLM-pilot data → never present in-silico results as human evidence (ethics §2).
